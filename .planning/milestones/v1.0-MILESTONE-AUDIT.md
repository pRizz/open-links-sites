## Residual Risks

These are not milestone blockers, but they should remain visible:

| Phase | Residual Risk | Impact |
| --- | --- | --- |
| `01` | Local asset references are translated at the materialization boundary | future upstream schema changes could require boundary adjustments |
| `03` | Linktree-style extraction is intentionally lightweight | some provider pages may need richer extraction later |
| `04` | Selective builds restore the live artifact before overlaying changed routes | deploy runtime can grow with artifact size |
| `05` | One shared release gate now covers sync and deploy | verification cost may increase as the repo grows |
